# LevelRelay

Link two levels together and switch between them from the pause menu or with a keybind. Auto SP Select can also pick the nearest startpos before where u died when switching into an SP copy.

## notes

* online levels are loaded only when u actually switch to them if gd doesnt already have the level loaded.
* switching scenes is delayed a tiny bit because doing it directly from pause can be unstable.
* Auto SP Select uses the closest startpos at or before your death x.
